Instruções de Instalação do Banco (CoderX)
Crie um novo projeto no Supabase.

Vá em SQL Editor e crie uma nova "Query".

Copie e cole o conteúdo do arquivo schema.sql e execute.

Vá em Storage, crie um Bucket público chamado logos.

Em Policies do bucket, habilite INSERT e UPDATE para usuários autenticados.

Copie as credenciais (URL e Anon Key) para o seu arquivo .env do projeto.
